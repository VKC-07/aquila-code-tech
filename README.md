# Aquila Code Tech

Simple static landing page for Aquila Code Tech's campus recruitment training offering.

Website: https://vkc-07.github.io/aquila-code-tech/

## Files

- `index.html` contains the page structure and content.
- `styles.css` contains the site styling.
- `script.js` contains the page JavaScript.

## Preview Locally

Open `index.html` in a browser.

## Deploy

This project can be hosted on any static hosting platform, including GitHub Pages, Netlify, or Vercel.
